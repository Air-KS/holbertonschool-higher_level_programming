Usign "add_integer" to Function
********************************

Importing the function from the modul
	>>> add_integer = __import__("0-add_integer").add_integer

Regular addition:
	>>> add_integer(2, 3)
	5

Positive with negative:
	>>> add_integer(2, -4)
	-2

Negative with positive:
	>>> add_integer(-6, 8)
	2
